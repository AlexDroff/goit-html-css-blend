## Description
A multi-page static website built with **HTML5**, **CSS3**, and minimal **JavaScript**, designed using a **mobile-first** approach to ensure full responsiveness across different devices and screen sizes.

The layout is implemented with **Flexbox** to achieve clean alignment, flexible spacing, and visual consistency across all pages. The project focuses on front-end performance, fast load times, and smooth user interaction, with no backend involved.

- Fully responsive, mobile-first layout
- Multi-page website with clear navigation
- Flexbox-based layout system
- Interactive mobile navigation menu
- Retina-optimized images
- Consistent design system (typography, spacing, colors)
- Semantic HTML for accessibility and SEO
- Optimized front-end performance
